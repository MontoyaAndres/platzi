# Tips

## Create new project

```shell
conda create --name NAME TOOLS

# Example:

conda create --name platzi_data beautifulsoup4 requests numpy pandas matplotlib yaml
```

## Start project

```shell
cd PROJECT_FOLDER
source active NAME

# Example:

source active platzi_data
```

## Remove project

```shell
conda remove --name NAME --all
```

## List enviroments

```shell
conda env list
```
