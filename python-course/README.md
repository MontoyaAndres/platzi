## Create virual env

```
# Install package
pip install virtualenv

# Create virtualenv (venv is the common one)
virtualenv venv

# Initialize
source venv/bin/activate
```

## Check packages installed

```
pip freeze
```

## Install packages by requirements.txt

```
pip install -r requirements.txt
```
